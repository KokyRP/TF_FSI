# Sistema de Gestión – Germania Automotriz

Sistema de información desarrollado como Trabajo Final del curso
**Fundamentos de Sistemas de Información (UPC)**.

El sistema permite gestionar de forma integral los procesos de atención y operación de un taller automotriz.

---

## Funcionalidades principales

- Gestión de Clientes y Vehículos  
- Gestión de Mecánicos y Servicios  
- Programación y control de Mantenimientos  
- Control de Inventario de Repuestos  
- Cálculo automático de cobranzas  
- Reportes de ingresos, demanda de servicios y repuestos  

---

## Tecnologías utilizadas

- C# (.NET Framework)  
- SQL Server  
- Arquitectura por capas  
- Git / GitHub  

---

## Enfoque en seguridad

- Validación de datos de entrada  
- Prevención de inyección SQL  
- Manejo de excepciones  
- Control de integridad de la información
